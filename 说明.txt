上传代码
1、新建文件夹，编写程序
git init
git add .
git commit -m "name"
2、将本地git仓库与远程联系起来
git remote add origin https://github.com/账户名/git仓库
3、上传代码
git push -u origin master
git push -u origin main

4、下载指定tag版本的代码
 git clone -b master git://github.com/qinguabao/appium_auto_test.git

5、合并代码
下载的新增的。
先拉取代码：git pull
6、查看日志及版本信息
git log
查看当前的commit号码
git rev-parse HEAD
7、新建分支
git clone -b [tag标志] https://github.com/aa/bb.git
# 然后切换
git checkout 2342dsfsdfs2
# 新建分支保存更改
git checkout -b branch
